# Stock Portfolio Tracker

##  Description

This project is a simple Python-based Stock Portfolio Tracker. It allows users to enter stock names and quantities, calculates the investment value, and displays the total portfolio value. The program also saves the result to a text file.

##  Features

* Input stock name and quantity
* Calculates investment for each stock
* Supports multiple stock entries
* Displays total investment
* Saves result to a `.txt` file

##  Concepts Used

* Dictionary
* Input/Output
* Loops
* Conditional Statements
* File Handling

## How to Run

1. Run the Python file:
   python stock_tracker.py
2. Enter stock name and quantity
3. Choose whether to continue
4. View total investment at the end

## Output

* Displays total investment in console
* Saves output in `portfolio.txt` file

## Example

AAPL → 2 shares
TSLA → 1 share
Total Investment: 610
